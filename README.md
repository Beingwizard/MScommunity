## 🧠 1. Artificial Intelligence (AI)

**🔍 Prerequisites:**
- Basic cloud knowledge  
- Interest in ML/data science  
- Python familiarity (for advanced learning)

**📘 Learn Modules:**
- [Get started with AI Fundamentals](https://learn.microsoft.com/en-us/training/modules/get-started-ai-fundamentals/)
- [Introduction to AI on Azure](https://learn.microsoft.com/en-us/training/paths/introduction-to-ai-on-azure/)
- [Fundamentals of Generative AI](https://learn.microsoft.com/en-us/training/modules/fundamentals-generative-ai/)
- [Design Regression Pipelines with Azure ML Designer](https://learn.microsoft.com/en-us/azure/machine-learning/tutorial-designer-automobile-price-train-score)

**🧠 Skills Covered:**
- ML basics (regression, classification)  
- NLP, computer vision, speech  
- Generative AI & prompting  
- Azure ML pipeline creation

**🧾 Certifications:**
- [Azure AI Fundamentals (AI‑900)](https://learn.microsoft.com/en-us/credentials/certifications/azure-ai-fundamentals/)
- Azure AI Engineer Associate  
- Azure Data Scientist Associate
